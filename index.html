1. Variables in Sass

Variables allow you to store values in a single place and reuse them throughout your stylesheet.

2. Nesting in Sass

Nesting allows you to write your CSS in a way that follows the same visual hierarchy of your HTML.

3. Mixins in Sass

Mixins allow you to create reusable styles. You can even pass in values to make them more dynamic.
